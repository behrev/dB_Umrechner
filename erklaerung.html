Grundlagen: Watt und Dezibel

In der Nachrichtentechnik und Audio-Elektronik wird das Verhältnis zwischen zwei Leistungswerten (Eingang vs. Ausgang) oft nicht linear (als Faktor), sondern logarithmisch in Dezibel (dB) angegeben.

Das hat zwei große Vorteile:

  • Sehr große und sehr kleine Zahlen lassen sich handlicher darstellen.
  • Verstärkungen von hintereinander geschalteten Stufen können einfach addiert statt multipliziert werden.
L = 10 · log₁₀(Pout / Pin)

Dabei ist Pin die Eingangsleistung und Pout die Ausgangsleistung.

Typische Werte (Faustformeln):

dB Wert Veränderung der Leistung
+3 dB Verdopplung (x 2)
+10 dB Verzehnfachung (x 10)
+20 dB Verhundertfachung (x 100)
-3 dB Halbierung (x 0.5)
0 dB Keine Änderung (x 1)
Bedienung des Programms

Dieses Programm dient dazu, schnell die Verstärkung oder Dämpfung zwischen zwei Messpunkten zu berechnen.

Schritt-für-Schritt:

  1. Eingangssignal festlegen:
    Geben Sie im linken Feld den gemessenen Wert ein. Wählen Sie im Dropdown-Menü, ob der Wert in Milliwatt (mW) oder Watt (W) vorliegt.

  2. Ausgangssignal festlegen:
    Geben Sie im rechten Feld den zweiten Wert ein. Auch hier können Sie die Einheit unabhängig vom ersten Feld wählen (das Programm rechnet dies intern um).

  3. Notizen (Optional):
    Sie können Stichworte (z.B. "Vorverstärker" oder "Antenne") in die Textfelder eintragen, um Ihre Berechnung zu dokumentieren (z.B. für Screenshots).

  4. Berechnen:
    Klicken Sie auf den Button "Berechnen". Das Ergebnis erscheint unten in der Leiste.

Hinweis: Positive dB-Werte bedeuten eine Verstärkung, negative Werte eine Dämpfung (Verlust).
